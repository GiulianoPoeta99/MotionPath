# tpi-gimnasio

## Iniciar submodulos

Este repositorio es el contenedor de otros 2 repositorios que conforman esta app. Para inicarlos hay que ejecutar los siguientes comandos en la terminal:

```bash
git submodule init
git submodule update
```
